데이터 입출력 방법

- JS를 통해 HTML이나 콘솔로 데이터를 출력하는 방법

데이터 출력법

1. document.write() 활용

- 브라우저(html 문서)로 문구를 출력하는 가장 기본적인 방법

2. window.alert('메세지')

- 브라우저 메세지창으로 출력하는 방법, window 생략 가능합니다.
- 사용성 측면에서는 좋지 않은 방법으로, 학습이나 디버깅 용도로 활용한다.

3. innerHTML, innerText ✭✭✭✭✭

- html로 작성된 태그를 찾아 객체 내부에 html이나 text를 출력하는 방법
- 일반적으로 id를 통해 탐색하여 태글르 조작하는 것이 표준적인 방법이다.
기본 문장

4. console.log()

- 개발자 도구 화면에 콘솔로 출력하는 방법
- 일반 사용자에게는 보여지지 않지만, 개발이 끝나면 정리하는 것이 바람직!
