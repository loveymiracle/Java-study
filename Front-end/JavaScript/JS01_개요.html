자바 스크립트의 개요

- 웹 브라우저에서 HTML문서(DOM)에 대한 동적 로직 처리가 가능한 인터프리터 방식의 언어
- 객체지향 문법을 지원하고, 실질적으로는 절차지향으로도 프로그래밍이 가능한 언어
- 실제 동작하는 방식은 브라우저에 존재하는 웹 엔진(크롬 엔진)을 통해 파싱 및 실행 가능

자바 스크립트의 선언방법

- HTML 파일 내에 script 태그를 활용하여 Javascript 영역을 생성함
- type="text/javascript"를 포함 해야 하지만 default가 js라 생략 가능


자바스크립트 코드의 위치

- 정석적으로는 header에 선언하거나 외부 파일로 link 시키는게 가장 좋은 방법이다.
- 페이지 내에서 어디든 가능하고 필요에 따라 사용자가 적절한 배치가 필요하다.

자바스크립트 작성 유형

1. internal(내부) 방식

- 일반적으로 html 내부에 작성하는 방식

2. inline 방식

- html 태그의 특정 속성(on~~~) 중 이벤트 핸들러 영역에 js를 직접 작성하는 방법

예시1 : 인라인 방식으로 HTML 속성 내부에 짧게 작성하는 방법

-> 아주 짧게 쓰는 것을 권장하고 주로 함수 호출을 통해 문자을 늘려서 활용

예시2 : 미리 선언한 함수를 통해 호출하는 방법 ★★★★★


3. External(외부)에서 호출하는 방법 ★★★★★

- 자바스크립트 코드를 외부에서 작성하여 로드하는 방식으로 가장 정석적!
- 단, js 코드 특성상 외부에서만 로드하는 것은 불가능하여 섞어서 사용한다.
- 파일 확장자는 주로 .js로 끝남 단, .js가 아니어도 로드는 가능하다.
