CSS 선택자란?

- 특정한 HTML 태그를 선택하여 스타일 부여할때 사용되는 식별자(태그, #id, .class, 후손, 자식 이벤트 등)

1. 모든 선택자

- HTML 문서안 모든 태그를 선택하는 기능

2. 태그 선택자 ★★

- HTML 문서안 특정 태그를 선택하는 방법
    ul 태그 영역입니다.
  • li 태그 영역입니다.

    • 3. 아이디 선택자(#) ★★★

    - HTML 태그의 속성값인 id를 통해 지정하는 방법

    - id와 style을 1:1로만 매핑할때 사용됨, 가끔 쓸 일이 있다.

    - id는 일반적으로 javascript에서 사용되는 식별자

    4. 클래스 선택자 ★★★★★

    - class와 style을 1:n으로 매핑하는 방법으로 1:1도 매핑도 가능, CSS 적용시 가장 많이 활용됨!
