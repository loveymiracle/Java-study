폼 관련 태그

form 태그

- 사용자가 입력한 값에 대한 처리 방식과 실제 처리할 서버의 url을 명시하는 태그

- action : 실제 서버가 처리해야 할 url/주소 정보를 기입

- method : get/post를 통해 서버에 전달 방식으로 기입 (default : get)

- submit : form의 입력된 데이터를 서버로 전달하는 트리거(버튼)

input 요소들이 써지는 곳.




fieldset, legend 태그 (나중가면 안쓰는 태그)

fieldset : 폼요소의 그룹을 묶어주는 태그

legend : 폼요소 그룹의 명칭을 붙여주는 태그
