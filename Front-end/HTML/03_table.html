표(table)관련 태그

Table 태그 ★★★★★

웹 문서에서 table 형태로 자료를 정리하기 위해 사용되는 태그
행과 열로 구성되어 있으며, 만나는 지점을 셀이라 한다.
table 태그 : 테이블을 지치하는 태그로 가장 밖에 선언된다.
tr 태그 : 한개의 행(row)을 나타내는 태그
td 태그 : 한개의 열을 나타내는 태그

테이블 예시

이름 나이
홍길동 21
최길동 34

border 속성 : 테이블의 테두리 굵기를 조절하는 속성, html4 문법 = 사용안하는 문법

이름 나이 전화번호
홍길동 21 010-1234-5678
최길동 34 010-4321-5678

caption 태그 : 테이블의 제목을 다는 태그
