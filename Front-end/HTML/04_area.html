div태그 ★★★★★ (Division)

- html문서에서 부분 또는 색션을 정의하기 위해 블록요소로 활용되는 태그
- 주로 다른 태그를 감싸는 컨테이너(요소안에 요소가 있는 형태)로 활용됨
- 대표적인 블록태그로 감싸면 한줄이 자동으로 개행 되는 효과 발생 (br)
- 디자인적인 요소는 거의 없는 무색무취 같은 태그 -> css를 적용하는 것이 일반적
- 주로 CSS나 javascript에서 조작하는 단위로 활용됨 -> id를 부여해서

span 태그 ★★★★★

- 문서의 인라인 요소를 그룹화하는데 사용하는 대표적인 inline 태그
- 단일 목적으로는 사용되지 않는 태그로 반드시 그룹화할때만 사용할 것 -> 단일 요소로 쓸모가 없다.
- 용도로는 주로 inline 영역에 css로 지정하거나 그룹핑 할 때만 사용한다.
- inline요소형으로 개행이 되지 않고 수평으로 확장할 때 사용한다.

span 응용3 - 높이/너비 지정해보기 - 되는 버전

- display 옵션을 통해 태그의 영역을 inline과 block 그리고 inline-block등 으로 변경할수 있다.
- inline-block 옵션은 인라인 요소인데, 너비나 높이가 지정 가능한 태그로 변경된다.
스탠은 인라인 영역입니다. 일반영역입니다.
