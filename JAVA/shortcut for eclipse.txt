Indent : 라인 들여쓰기. 고수 가장 중요하게 여긴다. tab 사용
Format : 띄여쓰기 ' '
tab : 들여쓰기
shift + tab : 역들여쓰
ctrl + space : 자동완성
ctrl + / : 주석 자동완성
ctrl + A : 모든 라인 선택
ctrl + i : 자동 들여쓰기
ctrl + shift + f : format 정렬 -> ctrl + i도 같이 수행
ctrl + z : 되돌리기
ctrl + y : 다시 실행하기
ctrl + d : 선택된 라인지우기
alt + ↑ : 선택된 라인 위로 옮기기
alt + ↓ : 선택된 라인 아래로로 옮기기
ctrl + alt + ↑ : 선택된 라인 위로 복사하기
ctrl + alt + ↓ : 선택된 라인 아래로 복사하기
